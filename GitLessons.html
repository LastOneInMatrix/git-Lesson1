1. git clone 'path to repo..'
  2. git status Узнать статус репо
  3. git add . Добавить файлы для трекинга гитом

    4. сделать изменения
  4. git commit -am 'Комментарий коммита' Зафиксировать изминения сделанные в файле входящим под контроль
  5. git push Отравить закомиченные файлы на удаленный репозиторий
  6. git pull Чтобы подтянуть изменения
